com.force.sdk.codegen.filter.ObjectNameDataFilter

Policy: default

Comment Text: A {@link DataFilter} which filters based on Force.com schema object (SObject) names.

This {@code DataFilter} will filter in a Force.com schema object if and only if its name exactly matches one in the {@code ObjectNameDataFilter} state. Names in the {@code ObjectNameDataFilter} state which match no schema objects are ignored.
