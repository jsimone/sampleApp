com.force.sdk.oauth.context.ForceSecurityContext

Policy: default

Comment Text: This is the default implementation of a SecurityContext. This is what will hold an authenticated user's data if a custom user data retriever is not used. When using a custom user data retriever the CustomSecurityContext will wrap a ForceSecurityContext and add the additional custom fields.
