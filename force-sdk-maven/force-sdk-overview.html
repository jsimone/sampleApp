What is the Force.com SDK for Java?

The Force.com SDK is a set of Java libraries and tools that makes it easy for Java developers to build applications that

  • use Force.com as the database for application data
  • interact with data already stored in Force.com
  • use the Force.com user management and security model
  • extend existing Force.com applications and functionality

The SDK is not tied to any particular runtime. You can build apps that run on your own server, on Cloud Foundry, on EC2 or other cloud runtimes as long as you can make HTTPS connections to Force.com.

The SDK consists of the following components

Force.com Service Connector

The service connector is a basic utility for connecting to the Force.com service. Its main responsibility is to read connection configuration from a variety of sources and to refresh sessions automatically. The connector uses WSC for the actual protocol implementation.

Read more

Force.com JPA Provider

The JPA provider lets you use Force.com as a persistent data store for entities defined in Java using the JPA standard. You can define entities in pure Java, including related entities using OneToMany or ManyToOne relationships and the JPA provider will automatically manage the schema in Force.com. It also lets you map existing entities in Force.com to JPA entities so you can query and transact on data that’s already in your Force.com database such as your CRM entities. The JPA provider is based on DataNucleus Access Platform.

Read more

Force.com OAuth authentication and authorization

If you’re building an application for authenticated Force.com users, you can configure single sign-on and leverage Force.com’s security model using OAuth2.

Read more

Spring security plugin

This plugin lets you use Spring Security for authentication and authorization in your applications. The Spring Security library uses the API Connector and the OAuth Connector to allow authentication via the Force.com APIs. There is also a custom Spring Security namespace that helps to simplify configuration.

Read more

CLIforce command-line tool

The CLIforce command-line tool is used to manage Force.com connections locally on you development machine and to manually execute tasks such as

  • creating new projects from templates,
  • updating the schema in Force.com and
  • importing data

The commands can be executed from an interactive shell or as batch scripts. You can also execute scripts via a maven plugin.

Read more

Spring MVC project template

The SDK includes a Spring MVC project template distributed as a Maven archetype. You can create new projects from this template using the cliforce template:create command. The template sets up all the rigth maven dependencies for starting a new Spring MVC app using the JPA provider and Spring Security plugin.
