com.force.sdk.oauth.userdata.CustomUserDataRetrievalService

Policy: default

Comment Text: Defines the logic which calls a CustomUserDataRetriever. When customer user data retrieval is used a CustomerUserDataRetriever that returns the CustomSecurityContext object will be needed. The CustomUserDataRetrievalService is called to retrieve custom user data and it will, in turn, call the application specific CustomerUserDataRetriever.
